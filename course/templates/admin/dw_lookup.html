Course Code
Subject Code + Course Number + Course Section. You may optionally include the Year + Term at the end if you wish to limit the results to a specific term. If you do not include the Year + Term, it will return courses from all periods matching the course subject, number, and section values. To allow for easy copy-and-pasting, the course code can be input in any form, whether than includes dashes, underscores, or spaces, or includes "SRS_" or not. Example: 'SUBJ-100-101' or 'SRS_SUBJ-100-101 2021A'.
